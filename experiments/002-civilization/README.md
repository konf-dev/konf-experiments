# Experiment 002: The Civilization

**Status: ACTIVE**

**Goal:** Transition from a central "push-based" orchestration to a decentralized "poll-based" agent bureaucracy.

**Hypothesis:** An "AI Bureaucracy" structured with an Onion Architecture (layered permissions) and a "Moral Compass" configuration can autonomously manage complex, cross-functional workflows without increasing the user's cognitive load.

## The Office of Offices (Vault)

The meta-documentation for this experiment is managed in a collaborative Obsidian vault located at `konf-meta-vault/`.

### Directory Structure
- **00_Shared/**: Core vision, brain dumps, and cross-agent knowledge.
- **10_Protocols/**: Ethics, monitoring specs, and communication standards.
- **20_Offices/**: Workspace for the AI Broker and specialized Research agents.
- **30_Registry/**: Job boards and agent capability registries.
